Red Hat | トレーニングスケジュール

このサイトでは、Red Hat が提供する予定の公式トレーニングの予定を始めとする情報を掲示しています。

  • SKUのリンクをクリックすると、そのトレーニングコースの概要ページにアクセスできます。
  • Offering IDのリンクをクリックすると、そのトレーニングセッションの申し込み用ページに移動します。

なお、2026年1月より Red Hat ではトレーニングの販売形式が年間サブスクリプション形式である「Red Hat Learning Subscription」というサービスに変更しました。

Red Hat Learning Subscription の詳細については こちらをご覧ください。

上記のリンクより1年間有効な学習サブスクリプションを購入後、1年間の間に任意の受講回でのトレーニング受講を頂きます。

このページのソースおよび docs/data.csv をはじめとするデータは、次の GitHub レポジトリで公開・管理されています。 Lab8010/redhat-training-scheduleです。
